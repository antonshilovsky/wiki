# Функциональные обязанности {#задачи}

| № | Функция | Основные действия | Достижения |
|---|----------|------------------|-------------|
| 1 | **BI-аналитика** | Создание и поддержка BI-систем, формирование витрин данных, настройка показателей. | Внедрено решение по скоринговой оценке контрагентов; построены дашборды в Power BI и Superset. |
| 2 | **ETL и подготовка данных** | Загрузка и валидация данных, оптимизация процессов отчётности. | Настроены СУБД ClickHouse и Access для 3000+ контрагентов. |
| 3 | **Интеграции и API** | Настройка REST API для выгрузки данных с сервиса Контур. | Настроен API-канал в СУБД. |
| 4 | **Анализ контрагентов** | Проведение проверки и скоринговой оценки партнёров. | Создана скоринговая модель с частичной автоматизацией скоринга — инженер самостоятельно загружает и обновляет данные.. |
| 5 | **Методическая работа** | Подготовка инструкций и регламентов в рамках разработки BI-системы. | Разработана методика создания цифровых курсов, внедрена инфраструктура Центра обучения сотрудников с нуля. |
| 6 | **Обучение и вебинары** | Проведение обучающих программ и консультаций по работе в соответствии с внутренними регламентами и в BI-системах. | Проведены корпоративные вебинары. |
| 7 | **Аналитика и отчётность** | Подготовка аналитических материалов и KPI для руководства. | Настроен API валютных курсов ЦБ РФ в бланке сделки. Решение внедрено в документооборот. |
| 8 | **Контроль качества данных** | Мониторинг корректности и полноты данных. | Настроен бэкап для витрин данных в BI. |
