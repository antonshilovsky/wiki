# Функциональные обязанности {#задачи}

*Порядок функций приведён по убыванию приоритетности — от аналитики контрагентов до задач BI и автоматизации.*

| № | Функция | Основные действия | Достижения |
|---|----------|------------------|-------------|
| 1 | **Проверка новых контрагентов** | Проведение проверки и скоринговой оценки партнёров, анализ финансовой отчётности и деловой репутации. | Создана математическая модель для автоматизации экспресс-скоринга. |
| 2 | **Аналитика и отчётность** | Подготовка аналитических материалов, формирование KPI и визуализация ключевых показателей для руководства. | Настроен API валютных курсов ЦБ РФ для внутренней системы учёта, решение внедрено в документооборот. |
| 3 | **Методическая работа** | Подготовка инструкций, регламентов и методик в рамках разработки BI-системы и внедрения в работу пользователей. | В корпоративном контуре внедрена инфраструктура Центра обучения и методика создания цифровых курсов с нуля. |
| 4 | **Обучение и вебинары** | Проведение обучающих программ, консультаций и корпоративных вебинаров для сотрудников. | Проведены вебинары по BI-системам и регламентам работы с данными. |
| 5 | **BI-аналитика** | Создание и поддержка BI-систем, проектирование витрин данных, разработка показателей. | Внедрено решение по скоринговой оценке контрагентов; построены дашборды в Power BI и Superset. |
| 6 | **Контроль качества данных** | Мониторинг корректности и полноты данных, настройка резервного копирования витрин. | Настроен бэкап для витрин данных в BI-системе. |
| 7 | **ETL и подготовка данных** | Загрузка, валидация и оптимизация процессов отчётности, построение структур данных. | Настроены СУБД ClickHouse и Access для данных контрагентов. |
| 8 | **Интеграция API в СУБД** | Настройка REST API для автоматической выгрузки данных с сервиса Контур.Фокус и внутренних источников. | Настроен API-канал в СУБД для ≈ 3000 наблюдений в год. |
